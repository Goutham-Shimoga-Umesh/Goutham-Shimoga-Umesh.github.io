Pentagon Building — Crosswind Loading (3D)

Urban bluff-body flow · pressure vs viscous drag contributions
3DBluff Body
Two configurations (Run 1, Run 2) · ANSYS Fluent · horizontal-plane diagnostics

Goal. Quantify how small geometric/placement changes around a pentagon-shaped building alter cross-wind wake structure and the breakdown of total drag into pressure and skin-friction components.

What I varied. Two runs with different local offsets/obstacles near the building (see sketch in report snapshot). Both cases were solved to steady convergence with the same inlet profile and domain size; results are compared on a mid-height horizontal plane.

Diagnostics. Static pressure and cross-stream velocity (v, Y-component) contours reveal near-wake recirculation, momentum deficit, and recovery rate downstream.

Setup (summary). Incompressible RANS; uniform approach flow; pentagon prism with sharp corners; walls no-slip; far-field boundaries sufficiently far to limit blockage. Reporting uses plane slices at the same elevation for both runs.

Interpretation: pressure drag dominates (>> skin friction), consistent with sharp-edged bluff-body separation. The small drop in total drag for Run 2 aligns with a shorter/wider recirculation region and faster pressure recovery.

Flow observations

  • Wake width & recovery. Run 2 shows a slightly broader but less intense deficit core on v (Y-velocity), suggesting faster lateral entrainment and earlier recovery of mean momentum.
  • Corner effects. High-gradient lobes at the windward corners (pressure) and shear layers (Y-velocity) are more symmetric in Run 2, indicating cleaner separation and reduced side-force bias.
  • Drag linkage. The smaller pressure deficit immediately behind the body in Run 2 correlates with the ~1–2% reduction in total drag seen in the report numbers.
Run 1 — Static pressure on the mid-height plane. Distinct low-pressure footprint aft of the windward face.
Run 1 — Y-velocity (cross-wind) on the same plane. Narrower, stronger deficit core indicates slower lateral recovery.
Run 2 — Static pressure. Slightly smoother pressure recovery behind the body vs Run 1.
Run 2 — Y-velocity. Broader wake footprint with reduced core intensity → earlier entrainment, marginally lower drag.

Takeaway. For sharp-edged urban forms, small layout changes can trim pressure drag by nudging separation and wake entrainment. Here, Run 2 achieves a modest drag reduction with a fuller near-wake and quicker pressure recovery.
